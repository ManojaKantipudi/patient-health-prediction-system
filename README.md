# Patient Health Prediction System

## Project Overview

This is a Flask-based web application that manages patient health records and predicts potential health risks using patient medical data.

## Features

* Add Patient Records
* View Patient Records
* Update Patient Records
* Delete Patient Records
* SQLite Database Integration
* Input Validation
* AI-Based Health Risk Prediction

## Technologies Used

* Python
* Flask
* SQLite
* SQLAlchemy
* Bootstrap 5

## Health Prediction Logic

The system predicts:

* High Diabetes Risk
* High Cholesterol Risk
* Possible Anemia
* Healthy

Based on:

* Glucose Level
* Haemoglobin Level
* Cholesterol Level

## Installation

```bash
pip install -r requirements.txt
python app.py
```
